# StudioFlow – Sistema de Gestão de Atendimentos 

O StudioFlow é um sistema web desenvolvido para gerenciamento de atendimentos, clientes e profissionais, com foco em organização, produtividade e controle administrativo. Foi criado como parte de um projeto integrador acadêmico, simulando um sistema real utilizado por estúdios de tatuagem para gestão de serviços.

---

## Funcionalidades

- Autenticação de usuários (login e cadastro)
- CRUD completo de clientes, atendimentos e profissionais
- Sistema de agendamento com visualização em calendário
- Dashboard com dados e métricas administrativas
- Histórico de atendimentos
- Filtros e busca de registros
- Interface responsiva e intuitiva

---

## Tecnologias Utilizadas

- Front-end: HTML, CSS, JavaScript  
- Back-end: PHP  
- Banco de Dados: MySQL 

---

## Estrutura do Projeto

O sistema foi estruturado com separação entre front-end e back-end, visando organização e escalabilidade.

- Organização de arquivos por responsabilidade
- Integração com banco de dados relacional
- Manipulação dinâmica de dados
- Estrutura preparada para expansão de funcionalidades
